 Nila is an online mental health platform helping users **find expert therapists, book appointments, and access support services** easily.

------

## Features
- Hero section with video and introduction  
- Therapist listing with profiles, experience, session duration, and pricing  
- FAQ section for common counselling questions  
- Appointment booking flow  
- Responsive UI built with Tailwind CSS  
- Built with React + Vite
------

## Tech Stack

- **Frontend:** React + TypeScript, Vite  
- **Styling:** Tailwind CSS  
- **Icons:** Lucide React  
- **Version Control:** Git & GitHub

## Clear Setup and Run Instructions
- **1.Clone the repository**
git clone https://github.com/IshvionTech/nila-website-frontend/tree/stebinsha-project
cd Nila-hospital

- **2.Install dependencies**
npm install
npm install lucide-react
npm install react-router-dom
npm install --save @fortawesome/fontawesome-svg-core @fortawesome/free-solid-svg-icons @fortawesome/react-fontawesome
npm install @fortawesome/free-brands-svg-icons

- **3.Run the development server**
npm run dev
